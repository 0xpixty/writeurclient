Команды для вкладок меню настроек

code CUIRect

book Это структура, представляющая прямоугольную область экрана в пикселях. Она имеет поля x, y (координаты верхнего левого угла), w (ширина) и h (высота). Используется для размещения элементов интерфейса.

Пример: git clone https://github.com/username/repo.git

code git add

book Добавляет изменения в индекс для следующего коммита.

Пример: git add index.html

code git commit

book Сохраняет изменения в локальном репозитории с сообщением.

Пример: git commit -m "Добавил новую страницу"
