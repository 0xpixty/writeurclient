Команды для вкладок меню настроек

code CUIRect SomeThing

book Это структура, представляющая прямоугольную область экрана в пикселях. Она имеет поля x, y (координаты верхнего левого угла), w (ширина) и h (высота). Используется для размещения элементов интерфейса.

book Переменные: Это экземпляры CUIRect, которые представляют подмодули (секции) внутри вкладки.

book Каждая переменная — это область, где будут рендериться определённые настройки. Эти области создаются путём разделения основной области (MainView) на части с помощью методов вроде VSplitMid или HSplitTop.

code VSplitMid

book Метод класса CUIRect, который делит прямоугольник вертикально пополам на две новые области.

Пример: MainView.VSplitMid(&OtherSettings, &OtherSettings2);

MainView — исходная область (обычно вся вкладка).

&OtherSettings — левая половина.

&OtherSettings2 — правая половина.

code DoLabel

book Метод для отображения текстовой метки в заданной области CUIRect. Принимает параметры: прямоугольник, текст, размер шрифта и выравнивание.

Пример: UI()->DoLabel(&Rect, "Settings", 20.0f, TEXTALIGN_CENTER);
